names = Array.new
names = []
names = ["Elan"]

See that up there? Those are just a few ways of making an array. It is really simple. Naming your array is the same as making a variable equal to your array. Now what is an array? An array is a collection of objects whether they be:

strings: "Elan",
fixed numbers/integers: 1, 2, 3,
float numbers/decimals: 1.0, 2.5, 3.2,
arrays: ["elan", "kvitko"], and more.

person = Hash.new
person = {}
person = {
:name => "elan",
:city => "nyc",
:favorite_food => "pizza"
:favorite_number => 15
}

Those things up there. They're what we call hashes in Ruby. A hash is another way of storing data except a hash works kind of like a dictionary. You see how I create a hash called person and if I wanted to represent myself as an object along with all of my attributes, I would use a hash. When I say a hash works like a dictionary, all it means is that there is a key which is the same as a word in a dictionary, and then you see the little hash rocket (what we call the equal sign with the greater than symbol. it's a cool word and pure genius!), which assigns whatever you put on the right of it the value which acts like a definition.
